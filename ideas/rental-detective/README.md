# AI 租房看房侦探

> **一句话卖点**：租房小白拍照合同，AI 3 秒标注风险条款，10 分钟看出房源猫腻，避免被坑

## 概述

基于 Issue #115 创建的详细设计文档。

租房是人生大事，但租房小白常常被中介话术忽悠、看房看不出问题、合同条款看不懂。**AI 租房看房侦探**通过拍照诊断 + 周边分析 + 合同扫描，让租房者在 10 分钟内全面了解房源真实情况，避免被坑。

## 功能设计

### 核心功能

1. **房源拍照诊断**
   - 拍房间照片，AI 自动识别并标注：
     - 渗水痕迹（墙面、窗台、天花板）
     - 裂缝（墙体、地板）
     - 采光情况（窗户朝向、室内亮度）
     - 插座数量和位置
     - 通风情况
   - 生成「房源健康报告」

2. **周边智能分析**
   - 基于定位分析真实通勤时间（步行/骑行/地铁）
   - 周边配套热力图（餐饮、医院、学校、超市）
   - 治安情况评估（基于公开数据 + 社区评价）
   - 噪音水平预测（临街、工地、地铁）

3. **租金价格对比**
   - 同区域同户型租金对比
   - 判断是否溢价及溢价幅度
   - 提供「合理租金区间」建议

4. **合同风险扫描**
   - 拍照或上传合同 PDF
   - 3 秒标注风险条款：
     - 隐藏费用（物业费、卫生费、服务费）
     - 不合理条款（提前解约违约金过高）
     - 模糊条款（押金退还条件不明确）
   - 提供修改建议

5. **看房清单生成**
   - 根据用户需求生成个性化检查清单
   - 支持场景：养猫、做饭、居家办公、健身、乐器
   - 自动提醒易忽略项

### 辅助功能

- **房源对比工具**：多套房源横向对比
- **谈判话术建议**：基于发现的问题提供砍价建议
- **历史记录**：保存所有看房记录，方便回顾

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| **竞品威胁**（链家/贝壳已有 VR 看房） | **差异化壁垒**：<br>1. **房源缺陷数据库**：积累真实房源缺陷数据，形成数据壁垒（链家/贝壳没有）<br>2. **用户信任品牌**：定位为「租房避坑神器」，建立中立、客观的品牌形象<br>3. **社区口碑**：建立用户社区，分享真实看房经验，形成网络效应<br>4. **技术优势**：专注于缺陷识别的 AI 模型训练，比通用平台更专业<br>**差异化定位**：「租房避坑神器」而非「看房工具」 |
| **数据合规**（爬取租房平台数据可能违规） | 使用「用户手动输入」或「合作 API」；建立数据合规团队 |
| **OCR 准确率低**（中文合同复杂） | GPT-4V + 人工模板混合识别；提供「人工审核」选项 |
| **GPS 室内不准** | 让用户手动标记位置或拍小区门口；结合地图 POI 校正 |
| **用户信任度低**（AI 诊断是否靠谱） | 提供「诊断依据」和「案例库」；引入「专家审核」机制 |
| **留存挑战**（租房是低频行为） | 扩展到「买房验房」场景；提供「房东版」工具；建立社区运营 |

## 隐私合规与数据处理

### 敏感信息处理方案

**合同扫描涉及敏感信息**（身份证号、银行账号、住址等），需要严格的数据保护措施：

| 处理环节 | 方案 |
|---------|------|
| **数据采集** | 1. 明确告知用户数据用途，获取授权<br>2. 提供「本地处理」选项（敏感信息不上传） |
| **数据传输** | 1. HTTPS 加密传输<br>2. 敏感字段脱敏后传输（身份证号中间位打码） |
| **数据处理** | 1. **优先本地处理**：OCR 和风险识别在用户设备上进行（边缘计算）<br>2. 云端处理时：数据加密存储，处理完立即删除<br>3. **数据匿名化**：合同分析时自动脱敏敏感信息 |
| **数据存储** | 1. **最小化存储**：只存储分析结果，不存储原始合同图片<br>2. 如需存储：加密存储 + 访问控制 + 审计日志<br>3. **数据保留期限**：用户可随时删除，30 天自动清理 |
| **数据安全** | 1. 符合《个人信息保护法》要求<br>2. 通过 ISO 27001 认证（长期目标）<br>3. 定期安全审计 |

### 用户隐私控制

- **查看权**：用户可查看所有存储的个人数据
- **删除权**：用户可一键删除所有数据（GDPR 标准）
- **导出权**：用户可导出个人数据
- **撤回同意**：用户可随时撤回授权，数据立即删除

### 合规声明

- 在产品中明确展示《隐私政策》和《数据处理协议》
- 合同扫描功能默认关闭，需用户主动开启并授权
- 提供「游客模式」：不保存任何数据，仅提供即时分析
