# AI 数字断舍离助手

> 智能分析数字生活，一键清理数字囤积

## 功能设计

### 核心功能
- **存储分析**：识别大文件、重复照片、过期下载
- **通知整理**：分析 App 推送习惯，建议关闭低价值通知
- **订阅清理**：检测长期未读的订阅/频道，建议退订
- **文件归类**：自动整理 Downloads/Desktop 混乱文件
- **断舍离报告**：每周生成清理建议，一键执行

### 差异化亮点
- 基于用户行为的智能决策系统（非简单清理）
- 「决策辅助」模式：推荐是否保留，用户最终确认
- 断舍离报告天然适合社交分享

## 技术方案

### 数据层
- **文件扫描**：Electron (Node.js fs) / Android Storage Access Framework
- **使用统计**：系统日志分析（Android UsageStatsManager）
- **订阅检测**：邮件解析 + 账单识别（Regex + LLM）

### 逻辑层
- **重复检测**：pHash/dHash 图片指纹 + SimHash 文本指纹
- **价值评估**：基于访问频率、文件大小、类型权重
- **决策引擎**：规则系统优先 + LLM 辅助解释（仅用户主动触发）

### LLM 使用场景
| 场景 | LLM 角色 | 调用频率 | 成本控制 |
|------|----------|----------|----------|
| 文件保留建议 | 解释理由，非决策 | 按需 | 本地模型优先 |
| 订阅价值分析 | 摘要邮件内容 | 每周一次 | 缓存结果 |
| 周报生成 | 个性化文案 | 每周一次 | 批量处理 |

**策略**：规则系统覆盖 80% 场景，LLM 仅用于「解释」和「复杂判断」

### 展示层
- **桌面端**：Electron（跨平台）
- **移动端**：先推 Android（系统权限更开放）
- **报告生成**：Markdown 模板 + 图表（Chart.js）

## 隐私与安全

### 数据处理原则
- **本地优先**：文件扫描、重复检测完全本地执行，不上传用户文件
- **最小权限**：仅申请必要权限，明确告知用途
- **用户可控**：随时可导出/删除所有数据

### 敏感数据处理
| 数据类型 | 处理方式 | 脱敏策略 |
|----------|----------|----------|
| 文件内容 | 本地分析 | 仅读取元数据，不上传 |
| 通知内容 | 本地存储 | 去标识化后分析 |
| 邮件内容 | 可选功能 | 仅提取订阅相关，不存储正文 |
| 使用统计 | 本地加密 | 匿名聚合后可上传 |

### 安全措施
- 所有本地数据 AES-256 加密存储
- 网络 API 仅用于可选的 LLM 服务，传输 TLS 加密
- 定期安全审计，支持第三方渗透测试
- 开源核心算法，接受社区监督

### 合规性
- 符合 GDPR 数据主体权利（访问、删除、携带）
- 中国用户数据境内存储
- 隐私政策透明，一键撤回授权
