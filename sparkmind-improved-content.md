# SparkMind AI - Creative Content Platform

## 📋 概述

SparkMind AI 是一个创新的AI驱动的创意内容平台，旨在为内容创作者、营销团队和企业用户提供全方位的创意内容生成、管理和优化服务。该平台整合了最先进的自然语言处理、图像生成和音频处理技术，帮助用户从创意构思到内容发布的全流程创作体验。

### 🎯 核心价值主张
- **一站式创意解决方案**：从文案到图像，从音频到视频，满足所有内容创作需求
- **AI辅助创作**：智能建议、自动生成、质量优化，提升创作效率
- **多平台内容管理**：统一管理分布在各平台的内容，实现跨平台发布
- **数据驱动优化**：基于用户反馈和数据分析，持续优化内容效果

### 数据库设计

#### 内容表
```sql
CREATE TABLE content (
    id SERIAL PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    content_type VARCHAR(50) NOT NULL,
    content_body TEXT,
    metadata JSONB,
    created_by VARCHAR(50),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    status VARCHAR(20) DEFAULT 'draft',
    is_published BOOLEAN DEFAULT false,
    published_at TIMESTAMP,
    view_count INTEGER DEFAULT 0,
    engagement_count INTEGER DEFAULT 0,
    conversion_count INTEGER DEFAULT 0
);
```

#### 创意表
```sql
CREATE TABLE ideas (
    id SERIAL PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    description TEXT,
    category VARCHAR(50),
    tags TEXT[],
    generated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    used_count INTEGER DEFAULT 0,
    success_rate DECIMAL(5,2) DEFAULT 0.00,
    ai_score DECIMAL(5,2),
    user_rating DECIMAL(5,2)
);
```

#### 用户平台连接表
```sql
CREATE TABLE user_platforms (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    platform_name VARCHAR(50) NOT NULL,
    platform_id VARCHAR(100) NOT NULL,
    access_token TEXT,
    refresh_token TEXT,
    expires_at TIMESTAMP,
    is_connected BOOLEAN DEFAULT true,
    connected_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    last_synced_at TIMESTAMP
);
```

#### 内容表现表
```sql
CREATE TABLE content_performance (
    id SERIAL PRIMARY KEY,
    content_id INTEGER REFERENCES content(id),
    platform_name VARCHAR(50) NOT NULL,
    metric_name VARCHAR(50) NOT NULL,
    metric_value DECIMAL(10,2) NOT NULL,
    recorded_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    date DATE NOT NULL
);
```
