# AI 面试模拟教练

> **一句话卖点**：应届生在面试前夜，用 AI 面试模拟教练语音对练3轮，从自我介绍都结巴到第二天对答如流拿offer

## 概述

基于 Issue #204 创建的详细设计文档。

面试是求职者最焦虑的环节。传统的准备方式（背题库、对镜子练、找朋友模拟）要么不够真实，要么反馈不专业。本产品通过 AI 语音对话模拟真实面试场景，根据用户投递的岗位 JD 自动生成定制问题，并在每次模拟后给出结构化反馈和改进建议。

## 功能设计

### 核心功能

- **岗位定制问题生成**：解析用户上传的 JD，自动生成匹配的面试问题
- **AI 语音面试官**：支持语音对话，真实还原面试场景，包括追问环节
- **实时反馈系统**：
  - 答案质量评分
  - 表达优化建议
  - STAR 法则引导
- **压力梯度训练**：从温和模式到压力面试，循序渐进提升抗压能力
- **复盘报告生成**：每次模拟后自动生成弱项分析和改进重点

### 辅助功能

- 历史模拟记录
- 常见问题题库（100+ 岗位模板）
- 导出复盘报告（PDF/Markdown）

## 技术方案

### 数据层

- **用户数据**：模拟记录、复盘报告、JD 解析缓存
- **题库数据**：100+ 岗位模板，支持用户自定义扩展
- **术语库**：支持用户上传专业术语，提升识别准确率

### 逻辑层

```
┌─────────────────────────────────────────────────────┐
│                    核心流程                          │
├─────────────────────────────────────────────────────┤
│  JD 上传 → 岗位解析 → 问题生成 → 语音对话 → 评估反馈  │
│     ↓          ↓          ↓          ↓         ↓    │
│  文本提取   关键词匹配   LLM 生成   Whisper    LLM    │
└─────────────────────────────────────────────────────┘
```

- **语音识别**：Whisper API（¥0.03/分钟）或本地 Whisper Tiny 实时预览
- **对话生成**：LLM + 结构化 Prompt（角色扮演 + 追问逻辑）
- **反馈生成**：STAR 法则模板 + LLM 评估

### 展示层

- **Web 端**：React/Vue + WebRTC 低延迟语音
- **移动端**：React Native 或小程序
- **输出格式**：Markdown 复盘报告，支持导出到 Notion/飞书

## 实现步骤

### Phase 1: MVP（2周）

1. 基础语音对话（Whisper API + TTS）
2. JD 解析 + 问题生成（5 个热门岗位模板）
3. 简单反馈（答案评分 + 1-2 条改进建议）

### Phase 2: 体验优化（2周）

4. 追问逻辑（根据回答深挖）
5. STAR 法则引导
6. 复盘报告生成

### Phase 3: 规模化（4周）

7. 扩展到 100+ 岗位模板
8. 压力面试模式
9. 历史记录和进度追踪
10. B2B 版本（企业定制）

## 变现模式

### B2C

- **按次付费**：¥19.9/次（面试场景低频但付费意愿强）
- **订阅制**：¥49/月，无限模拟 + 全部岗位题库

### B2B

- 与招聘平台合作（BOSS直聘、猎聘）作为增值服务
- 企业培训版本（定制岗位题库 + 团队管理）

### 扩展场景

- 职场晋升（晋升答辩、绩效汇报）
- 提高使用频次，提升 LTV

## 风险与缓解

| 风险 | 缓解措施 |
|------|----------|
| 语音识别受紧张情绪影响（口吃/停顿） | 增加语音预处理，过滤填充词；支持文字输入备选 |
| 追问逻辑机械感强 | 设计多轮追问模板，模拟真实面试官风格 |
| 低频场景导致流失 | 扩展到职场晋升场景，提高使用频次 |
| ChatGPT 免费替代 | 强调定制化岗位问题 + 专业反馈报告的价值 |
| 效果验证难 | 收集用户反馈，展示「模拟后拿到 offer」的案例 |
