# AI 口语陪练伴侣 - 随时随地练习真实对话

> 让 AI 成为你的 24/7 外语陪练，场景化对话 + 实时纠音，从此告别「不敢开口」

## 功能设计

### 核心功能
1. **场景化对话练习** - 点餐、问路、面试、商务谈判等真实场景模拟
2. **实时语音纠错** - 发音、语调、用词即时反馈，不打断对话流
3. **自适应难度引擎** - 根据用户水平自动调整词汇复杂度和语速
4. **文化语境贴士** - 地道表达建议，避免文化误解
5. **学习进度追踪** - 可视化练习时长、纠错趋势、场景覆盖

### MVP 范围（精简版）
- **2 个核心场景**：餐厅点餐、自我介绍（先验证核心流程）
- **中英双语**支持（后续扩展日/韩/西）
- **Web 端优先**（移动端作为 Phase 2）

## 技术方案

### 数据层
```
用户数据:
  - 语音录音 (OSS/S3)
  - 练习历史 (PostgreSQL)
  - 发音基线 (向量数据库)

场景数据:
  - 对话模板 (JSON Schema)
  - 文化知识库 (RAG)
  - 难度标签 (分级系统)
```

### 逻辑层
```
语音处理流水线:
  ASR (Whisper) → 意图识别 → 对话管理 → 响应生成 → TTS (Edge TTS)
       ↓
  发音评估 (见下方详细方案)
       ↓
  纠错建议生成 (LLM)
```

### 发音评估技术方案（MVP 明确方案）

**方案选择：对接 Azure Speech SDK**

| 方案 | 优点 | 缺点 | 决策 |
|------|------|------|------|
| Azure Speech SDK | 成熟稳定、准确度高、有发音评估 API | 按调用量计费 | ✅ MVP 采用 |
| 自研模型 | 零边际成本 | 开发周期长、需大量数据 | Phase 2 考虑 |
| 开源方案 (Kaldi) | 免费 | 需要语音专家调优 | 不推荐 |

**Azure 发音评估 API 返回：**
- `AccuracyScore` - 音素准确度 (0-100)
- `FluencyScore` - 流利度 (0-100)
- `CompletenessScore` - 完整度 (0-100)
- `PronunciationScore` - 综合发音分数

**成本估算：** 约 $0.25/1000 次评估，MVP 阶段可控

### 实时纠错交互设计

**设计决策：对话后统一反馈**

| 模式 | 优点 | 缺点 | 决策 |
|------|------|------|------|
| 实时打断 | 即时纠正 | 打断用户思路、体验差 | ❌ 不采用 |
| 对话后反馈 | 不打断对话流 | 反馈延迟 | ✅ MVP 采用 |
| 混合模式 | 兼顾两者 | 实现复杂 | Phase 2 考虑 |

**具体交互：**
1. 用户完成一轮对话（2-4 轮）
2. 系统生成「学习报告」卡片：
   - 发音得分（可视化雷达图）
   - 错误 Top 3（带音频对比）
   - 改进建议（AI 生成）
3. 用户可选择「重新练习」或「进入下一场景」

### 移动端方案

**MVP 阶段：聚焦 Web 端**
- 响应式设计，移动浏览器可访问
- PWA 支持，可添加到主屏幕

**Phase 2 方案：React Native**
- 共享 80% 业务逻辑
- 原生语音 API 更稳定
- 推送通知提升留存

## 文化知识库数据来源

### RAG 数据来源（Phase 2 前）

| 来源 | 内容 | 获取方式 | 质量保障 |
|------|------|----------|----------|
| 维基百科 | 文化习俗、礼仪 | 官方 API | 人工抽检 |
| 语言学习博客 | 地道表达、俚语 | RSS 聚合 + 清洗 | AI 初筛 + 人工复核 |
| 用户贡献 | 场景化表达 | 众包功能 | 社区投票 |
| 专家审核 | 高价值内容 | 外教顾问 | 专业审核 |

**数据处理流程：**
1. 爬取/导入原始数据
2. LLM 提取结构化知识（场景 + 表达 + 文化背景）
3. 人工抽检 10% 样本
4. 入库向量数据库 (Pinecone/Milvus)

## 商业模式

### 变现路径
- **Freemium**: 2 场景免费，¥19.9/月解锁全部 + 高级反馈
- **课程捆绑**: 与雅思/托福机构合作导流
- **企业版**: 外企员工口语培训 ¥99/月

### 竞品深度分析

| 竞品 | 核心痛点 | 我们的优势 |
|------|----------|-----------|
| **Duolingo** | 游戏化练习脱离真实对话场景 | 真实场景模拟，直接可用 |
| **HelloTalk** | 依赖真人语伴，时间难协调、质量不稳定 | AI 24/7 可用，稳定一致 |
| **Speak** | 网络限制，国内访问不稳定 | 国内服务器，流畅体验 |
| **Cambly** | 价格昂贵（¥100+/小时） | 性价比高（¥19.9/月无限） |

**用户痛点洞察：**
- 60% 学习者「不敢开口」→ 非评判性的 AI 陪练
- 40% 学习者「没时间」→ 碎片化 5 分钟练习
- 35% 学习者「找不到语伴」→ 24/7 AI 可用

## 风险与对策

| 风险 | 对策 |
|------|------|
| 用户担心「纠错太多」 | 正向反馈优先，纠错比例可调（设置中选择「温和/标准/严格」） |
| 付费转化不确定 | 先 MVP 免费 + 邀请制内测，积累种子用户 |
| Azure 费用超预期 | 设置每日调用量上限，超出降级为基础反馈 |
| 发音评估准确度争议 | 保留用户反馈入口，持续优化模型 |
