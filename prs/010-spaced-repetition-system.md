# AI 遗忘曲线复习助手 - 科学记忆强化系统

> 基于艾宾浩斯遗忘曲线的智能复习系统，在最优时间点推送个性化复习内容，让记忆效率最大化。

## 功能设计

### 核心功能
- **智能学习追踪**：记录用户学习每个知识点的初始时间和复习历史
- **个性化遗忘曲线**：根据用户反馈（记得/模糊/忘了）动态调整复习间隔
- **精准时间提醒**：基于遗忘曲线算法在最优时间点推送复习提醒
- **多样化复习形式**：AI 自动生成填空题、选择题、简答题等不同形式的复习内容
- **知识关联强化**：自动识别关联知识点，形成知识网络，强化长期记忆

### 扩展功能
- **学习数据分析**：可视化记忆曲线、复习效率统计
- **社交化学习**：分享学习成果、好友 PK 记忆力
- **跨平台同步**：支持多设备学习进度同步

## 技术方案

### 数据层
```
用户学习数据（本地优先）
├── 学习记录：知识点 ID、初始学习时间、复习历史
├── 记忆曲线：个性化参数（记忆强度、遗忘速率）
├── 复习内容：AI 生成的题目库
└── 统计数据：复习次数、正确率、记忆保持度
```

**存储方案**：
- **本地存储**（隐私友好）：IndexedDB / SQLite
- **可选云端同步**：加密存储到云端，支持多设备

### 逻辑层

**核心算法 - SM-2+ 改进版**：
```
下次复习间隔 = f(
  上次间隔,
  用户反馈质量（0-5）,
  知识点难度,
  个性化遗忘参数
)

基础间隔序列：1天 → 3天 → 7天 → 15天 → 30天 → 60天 → 120天
根据反馈动态调整：记得（+20%间隔）/ 模糊（维持）/ 忘了（重置）
```

**AI 题目生成流程**：
1. 提取知识点核心概念
2. LLM 生成多种题型（填空、选择、简答）
3. 验证题目质量（答案唯一性、难度适中）
4. 缓存生成结果，避免重复调用

### 展示层

**交互流程**：
```
用户学习新知识 → AI 提取关键点 → 生成初始复习内容
       ↓
设置首次提醒（1天后）
       ↓
用户复习 → 反馈记忆状态 → 调整下次复习时间
       ↓
循环迭代，形成长期记忆
```

**通知渠道**：
- 优先级 1：Telegram / Discord Bot
- 优先级 2：邮件通知
- 优先级 3：App 推送

## 资源需求

### API 服务
- **LLM API**：用于生成复习题目
  - 推荐：OpenAI GPT-4o-mini（性价比高）
  - 备选：Claude Haiku、智谱 GLM-4-Flash
  - 预估成本：¥0.01-0.05/知识点

- **通知服务**：
  - Telegram Bot API（免费）
  - SendGrid / Resend（邮件通知）

### 基础设施
- **后端服务**：
  - Serverless 函数（Vercel / Cloudflare Workers）
  - 预估成本：免费层足够 MVP

- **数据库**：
  - 本地存储（用户设备）
  - 可选：Supabase / PlanetScale（云端同步）

### 开发资源
- **前端**：React + TypeScript + Tailwind CSS
- **后端**：Node.js / Python
- **预计开发时间**：8-10 周（MVP + 核心功能）
