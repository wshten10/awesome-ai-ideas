# AI 二手物品定价师

> **一句话卖点**：闲鱼卖家拍张照片，AI 3秒给出精准估价 + 生成吸睛描述，从纠结3天到10分钟上架卖出

## 概述

基于 Issue #162 创建。解决闲鱼/转转卖家的核心痛点——不知道该定多少钱，定高了卖不掉，定低了亏大了。

## 功能设计

### 核心功能
- **拍照估价**：拍物品照片 → AI 分析品相（全新/99新/95新/战损版）→ 3秒给出价格区间
- **智能定价**：基于近期成交数据（7/30/90天）给出「急出价」vs「等待价」
- **描述生成**：自动生成标题 + 卖点 + 关键词，提升曝光率
- **价格追踪**：监控同类商品价格变化，动态调价建议

### 辅助功能
- **历史成交参考**：展示 10 条同类商品成交记录
- **平台对比**：闲鱼/转转/小红书价格差异分析
- **上架提醒**：最佳上架时间推荐（周末/晚间高峰）

## 技术方案

### 数据层
- **价格数据库**：PostgreSQL 存储历史成交数据
- **向量数据库**：Milvus/Pinecone 存储商品特征向量（相似商品检索）
- **爬虫服务**：定时抓取闲鱼/转转成交数据（需解决反爬）
- **缓存层**：Redis 缓存热门商品价格

### 逻辑层
- **多模态识别**：GPT-4V/GLM-4V 识别物品类别、品牌、品相
- **价格预测模型**：
  - 协同过滤（相似商品定价）
  - 时间序列分析（价格趋势）
  - 规则引擎（品牌溢价、季节因素）
- **描述生成**：LLM 生成吸睛标题 + 卖点描述

### 展示层
- **移动端优先**：微信小程序 + App
- **核心交互**：拍照 → 3秒估价 → 一键上架
- **可视化**：价格走势图、成交概率预测

## 实现步骤

### Phase 1：MVP（2周）
1. **单品类验证**：聚焦手机/数码品类
2. **基础估价**：拍照 → 品牌识别 → 基础估价
3. **描述生成**：自动生成标题 + 3 个卖点
4. **冷启动数据**：手动录入 1000 条成交数据

### Phase 2：多品类扩展（4周）
1. 扩展至潮鞋/包包/图书/家电
2. 多平台数据整合（闲鱼+转转+小红书）
3. 智能定价策略（急出价 vs 等待价）
4. 价格追踪 + 动态调价建议

### Phase 3：商业化（4周）
1. 批量估价功能（专业卖家）
2. 店铺版（多账号管理）
3. 与回收平台合作（B2B2C）
4. 数据服务（向品牌方提供二手市场行情）

## 资源需求

### API
- **多模态识别**：GPT-4V / GLM-4V（¥0.02/次）
- **LLM 生成**：kimi-k2.5 / GLM-4（¥0.01/次）
- **地图/定位**：高德 API（用于地区差价分析）

### 服务
- **数据库**：PostgreSQL（阿里云 RDS）
- **向量数据库**：Milvus（自建）或 Pinecone（托管）
- **爬虫服务**：Scrapy + 代理池

### 数据
- **冷启动数据**：1000 条/品类 × 10 品类 = 10,000 条
- **持续更新**：每日 10,000 条新增成交数据

## 变现模式

### C 端
- **Freemium**：基础估价免费，深度报告 ¥9.9/次
- **订阅制**：¥29.9/月，无限估价 + 价格追踪 + 最佳上架时间
- **按效果付费**：成交后抽成 5%（帮用户多卖 100 元，收 5 元）

### B 端
- **专业卖家版**：¥199/月，批量估价 + 店铺管理 + 数据分析
- **平台合作**：与闲鱼/转转官方合作，作为估价工具接入
- **数据服务**：向品牌方提供二手市场行情报告（¥999/年）

## 风险与缓解

| 风险 | 缓解措施 |
|------|---------|
| 平台反爬 | 优先官方 API，建立数据合作；用户众包数据作为补充 |
| 数据准确性 | 价格波动大时降低置信度；展示「同类商品成交价」作为依据 |
| 用户信任 | 提供「估价依据」（10 条成交记录）；建立反馈闭环 |
| 竞品跟进 | 建立数据壁垒（成交价格数据库）；品牌/品相识别模型 |
| 品类扩展难度 | 先聚焦高频品类（手机/潮鞋）；逐步扩展 |

## 竞品分析

| 竞品 | 优势 | 劣势 | 我们的差异化 |
|------|------|------|------------|
| 闲鱼估价工具 | 官方数据 | 不透明、功能单一 | 详细报告 + 描述生成 |
| 爱回收 | 专业 | 只收不卖 | 估价 + 上架一体化 |
| 通用 ChatGPT | 免费 | 不懂市场行情 | 基于真实成交数据 |
| 小红书搜索 | 真实案例 | 需要手动筛选 | AI 自动分析 |

## 成功指标

### 北极星指标
- **GMV 助力**：帮助用户多卖的总金额（目标：1000 万/月）

### 关键指标
- **估价准确率**：估价与实际成交价偏差 < 15%（目标：80%）
- **用户留存**：7 日留存 > 40%
- **转化率**：估价 → 实际上架 > 60%
- **NPS**：净推荐值 > 50
