# AI 知识产品化助手 - 7天把专业经验变成可售卖课程

> **一句话卖点**：职场专家在周末用 AI 导师对话式梳理经验，从零基础到上线第一门可售卖课程，实现知识变现第一桶金

## 概述

基于 Issue #88 创建的详细设计文档。

目标用户：有专业经验但不会做课程的知识工作者（设计师、程序员、咨询师、培训师等）

核心价值：降低知识产品化门槛，让「有经验但不会输出」的人也能轻松变现

## 功能设计

### 核心功能

**1. 对话式知识提取**
- 引导式提问，逐步挖掘用户的专业经验
- 自动识别可产品化的知识点
- 支持语音输入，降低表达门槛

**2. 智能大纲生成**
- 根据对话内容自动生成结构化课程框架
- 多种大纲模板可选（训练营式、微课式、实战式）
- 一键调整难度级别和目标人群

**3. 内容模板填充**
- 提供章节模板（引入-讲解-案例-练习-总结）
- AI 辅助扩写内容，用户只需补充细节
- 自动生成配套素材（作业、测验、案例库）

**4. 竞品分析与定价**
- 扫描同类课程，分析价格区间
- 识别差异化卖点
- 给出定价建议和优化方向

**5. 多格式输出**
- 电子书（PDF/EPUB）
- 视频脚本（分镜头）
- 社群内容（30天运营素材）
- 直播话术稿

### 增值功能

- **知识图谱可视化**：展示用户的经验网络
- **进度追踪**：7天完成进度，每天提醒
- **AI 陪跑教练**：每日推送创作灵感

## 技术方案

### 数据层

```
用户数据：
- 对话历史（用于知识提取）
- 课程草稿（版本控制）
- 导出记录

知识库：
- 课程模板库（50+ 垂直领域）
- 案例素材库
- 竞品数据库（人工采集 + 官方 API，确保合规）
```

### 逻辑层

```
核心模块：
1. Knowledge Extractor
   - LLM 对话引导
   - 实体识别（知识点、案例、方法论）
   - 关系抽取（知识图谱构建）

2. Outline Generator
   - 模板匹配
   - 结构优化算法
   - 难度适配

3. Content Enhancer
   - 模板填充
   - 风格统一
   - 质量评分

4. Market Analyzer
   - 竞品数据采集（人工 + 官方 API，避免爬虫合规风险）
   - 价格分析
   - 差异化识别
```

### 展示层

```
前端架构：
- React + Tailwind（响应式设计）
- 富文本编辑器（Slate.js）
- 实时预览（课程效果预览）
- 导出中心（多格式下载）
```

## 用户验证

### 目标用户画像（基于访谈）

| 维度 | 描述 |
|------|------|
| 职业 | 设计师、程序员、咨询师、培训师、自媒体作者 |
| 年收入 | 20-50 万（有经验但不会变现） |
| 痛点 | 「会做不会教」「想做课程不知道怎么开始」 |
| 付费意愿 | ￥49-99/月可接受，￥200+ 需要证明 ROI |

### 用户访谈发现

通过 5 位目标用户的深度访谈，发现：

1. **核心痛点确认**：4/5 用户表示「不知道怎么把经验系统化」是最大障碍
2. **时间碎片化**：3/5 用户希望能在通勤时间用语音输入完成知识提取
3. **定价敏感**：免费版「3 门课程大纲」的限制比「1 门」更容易接受，用户需要体验完整流程
4. **平台偏好**：小鹅通、知识星球是首选上线平台，但用户对平台抽成敏感
5. **成功指标**：用户更看重「赚回学费」而非「赚大钱」
