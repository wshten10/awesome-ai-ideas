# DeceptionResilient AI - Advanced Training Platform for Critical Reasoning under Uncertainty

## 📋 概述

DeceptionResilient AI 是一个综合性的AI驱动训练平台，专注于在不确定性和欺骗条件下培养批判性思维能力。该平台基于多智能体系统和欺骗检测的前沿研究成果，为专业人士提供高度定制化的认知训练解决方案。

### 🎯 核心价值主张
- **对抗欺骗环境**：通过AI模拟各种欺骗场景，训练用户识别和应对虚假信息
- **提升认知韧性**：培养在压力和信息不对称环境下的决策能力
- **跨域应用**：支持网络安全、高管决策、医疗诊断、法律推理等多个专业领域
- **个性化训练**：根据用户表现和能力水平动态调整训练难度和内容

### 数据库设计

#### 用户表
```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    full_name VARCHAR(100),
    avatar_url VARCHAR(255),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    last_login_at TIMESTAMP,
    subscription_tier VARCHAR(20) DEFAULT 'free',
    preferences JSONB
);
```

#### 训练场景表
```sql
CREATE TABLE training_scenarios (
    id SERIAL PRIMARY KEY,
    title VARCHAR(200) NOT NULL,
    description TEXT,
    domain VARCHAR(50) NOT NULL,
    difficulty_level INTEGER NOT NULL,
    estimated_duration INTEGER, -- 分钟
    content JSONB NOT NULL,
    created_by VARCHAR(50),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    is_active BOOLEAN DEFAULT true
);
```

#### 训练会话表
```sql
CREATE TABLE training_sessions (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    scenario_id INTEGER REFERENCES training_scenarios(id),
    status VARCHAR(20) DEFAULT 'in_progress',
    start_time TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    end_time TIMESTAMP,
    score DECIMAL(5,2),
    completion_percentage INTEGER DEFAULT 0,
    data JSONB
);
```
